# 一行目にコントロールするロボットの名前を ; 区切りで列挙、３行目以降にシナリオを記述
# シナリオの書式は
# 開始秒 : ロボット名: コマンド, ロボット名: コマンド, ..
# コマンドは Animated Say ボックスに渡されるテキストです。 たとえば orange が「こんにちは」と言った後
# blue が手を振りながら「こんにちは」という。また５秒後に white が「こんにちは」というというシナリオは次の
# ように書きます
#
# 0.0: orange: こんにちは, blue: ^start(animations/Stand/Gestures/Hey_1) こんにちは ^wait(animations/Stand/Gestures/Hey_1)
# 5.0: white: こんにちは
#
# また - 記号は一つのストーリーの終了を意味します。ここまでのコマンドが全て終了してから次の行の処理が
# 進みます。次の行からの時間は再び 0 行から始まります。
